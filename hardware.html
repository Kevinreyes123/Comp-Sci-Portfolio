Mother Board

The mother board is the most important part of a computer because it sends signals to the cpu and ram. The mother board is like a comunication system that alows the ram and cpu to talk to eachother. The mother board is basaclly a circut board.

CPU

The CPU is also known as the Central proccessing Unit. The CPU carries out instructions to the other parts inside the computer. It is just as important as the motherboard.

Ram

Ram is "random memory". This means that it stores memory but is stored at the currsnt time and later wiped once the computer is turned off. Adding lots of ram will make your computer faster so people tend to buy lots of it.

Hard Disk Drive

The hard disk drive is a place to store memory for a long period of time and it contains more space than a ram stick. The memory will never be deleted unless the user deletes the memory that is stored.
